What is this?

Learn how to add and subtract mixed fractions by playing a variation on the battleship game.

Game Controls

How to Play
  1. Print out this web page. Adjust the scale so it prints as exactly two pages.
  2. Each player gets a page.
  3. Each player draws their battleships in their top grid (ocean).
  4. Each player takes turns picking a cell from their bottom grid (radar).
  5. Solve the problem in the cell and announce the cell location and the solution.
  6. The other player replies "Targeting Error" if the solution given doesn't match the answer in their top grid.
  7. If the solution does match the answer, the other player replies "Hit" or "Miss" depending on whether the cell contained part of a ship.
  8. If all of the cells for a ship have been hit, the other player announces that their ship is sunk.
  9. Continue taking turns until one player loses all their ships.
Tips
  • Refresh this page to generate new problems and answers.
  • If you print multiple pages, make sure the "Game seed" matches for both players.
